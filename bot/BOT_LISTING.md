# FilingsFlow Bot - Discord Bot Listing

## Short Description (up to 150 chars)
Real-time SEC insider trading alerts. Track Form 4 filings, 13F holdings, and congressional trades directly in Discord.

## Long Description

### What is FilingsFlow?

FilingsFlow delivers **real-time SEC filing intelligence** directly to your Discord server. Track insider buying and selling, see which hedge funds are accumulating positions, and monitor congressional stock trades - all without leaving Discord.

### Features

**Insider Trading Alerts**
- Real-time Form 4 filing notifications
- See when executives buy or sell shares
- AI-generated summaries explain each transaction
- Cluster detection when multiple insiders trade together

**Institutional Tracking**
- 13F holdings data from major hedge funds
- See which funds are buying/selling your favorite stocks
- Quarterly position change tracking

**Congressional Trades**
- Track stock trades by members of Congress
- House and Senate transaction monitoring
- Search by ticker or member name

**Watchlists**
- Add tickers to your server's watchlist
- Get alerts only for stocks you care about
- Configurable notification settings

### Commands

- `/insight <ticker>` - Quick snapshot of insider activity and news
- `/insiders <ticker>` - Recent insider transactions
- `/whales <ticker>` - Top institutional holders
- `/congress ticker <symbol>` - Congressional trades for a stock
- `/congress member <name>` - Trades by a specific member
- `/watchlist add/remove/list` - Manage your watchlist
- `/invite` - Get a link to add FilingsFlow to your server

### Why FilingsFlow?

- **Free tier available** - 5 watchlist tickers, 24-hour delayed data
- **Real-time processing** - Filings appear within minutes of SEC publication
- **No spam** - Only get alerts for tickers you're watching
- **Transparent** - Direct links to SEC EDGAR for source verification
- **Not investment advice** - Just the data, no hype

### Getting Started

1. Invite FilingsFlow to your server
2. Use `/watchlist add AAPL` to start tracking
3. Get notified when insiders trade
